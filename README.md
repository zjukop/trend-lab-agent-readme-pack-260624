# Agent README Pack

A tiny tool that turns any GitHub repo into an optimized onboarding manual for Codex, Claude Code, Cursor, and other agents—so maintainers can add one badge and make their project instantly agent-friendly.

## Why This Repo

This project is generated by a trend-driven pipeline. It targets **Open-source maintainers, AI coding-agent power users, developer educators, and teams adopting Codex/Claude/Cursor** and focuses on the niche: **AI coding-agent onboarding and context compression for existing codebases**.

## Key Features

- One-command scanner that reads a repo and generates a compact AGENTS.md/CODEX.md/CLAUDE.md context pack
- Token-budgeted summaries for architecture, commands, tests, conventions, dependencies, and risky files
- Works offline with tree-sitter/static heuristics, with optional LLM provider for higher-quality summaries
- Outputs a shareable HTML/PDF “repo onboarding book” inspired by popular Codex guide formats
- GitHub Action that auto-refreshes agent context files on PRs and comments when they are stale
